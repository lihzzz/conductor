# Conductor 测试场景与操作流程

## 测试环境准备

### 前置条件

1. **安装 Claude CLI**
   ```bash
   # 确认 claude 命令可用
   which claude
   claude --version
   ```

2. **创建测试 workspace**
   ```bash
   mkdir -p ~/test-conductor-workspace
   cd ~/test-conductor-workspace

   # 创建一些测试文件
   echo "print('Hello from test')" > main.py
   echo "def add(a, b): return a + b" > utils.py
   echo "# Test Project" > README.md
   ```

3. **构建并启动应用**
   ```bash
   cd /Users/lh/code/conductor
   npm run build
   npm start
   ```

## 自动化测试脚本

```bash
#!/bin/bash
# test-conductor.sh

set -e

echo "=== Conductor 自动化测试 ==="

# 检查 claude 命令
if ! command -v claude &> /dev/null; then
    echo "❌ claude CLI 未安装"
    exit 1
fi
echo "✅ claude CLI 已安装"

# 检查构建
if [ ! -d "dist" ]; then
    echo "❌ 项目未构建，运行 npm run build"
    exit 1
fi
echo "✅ 项目已构建"

# 创建测试 workspace
TEST_WS=~/test-conductor-workspace
mkdir -p "$TEST_WS"
echo "print('hello')" > "$TEST_WS/main.py"
echo "✅ 测试 workspace 已创建: $TEST_WS"

# 检查数据库
DB_PATH=~/.conductor/conductor.db
if [ -f "$DB_PATH" ]; then
    echo "⚠️  数据库已存在: $DB_PATH"
    read -p "是否删除并重建？(y/n) " -n 1 -r
    echo
    if [[ $REPLY =~ ^[Yy]$ ]]; then
        rm -f "$DB_PATH"
        echo "✅ 数据库已删除"
    fi
else
    echo "✅ 数据库不存在，将自动创建"
fi

echo ""
echo "=== 测试准备完成 ==="
echo ""
echo "运行以下命令启动应用:"
echo "  npm start"
echo ""
echo "然后在应用中:"
echo "  1. 输入 workspace: $TEST_WS"
echo "  2. 点击 Create Team"
echo "  3. 点击 Start All"
echo "  4. 在终端中输入命令测试"
```
